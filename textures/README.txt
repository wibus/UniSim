No data is stored on the git repository.

The textures need to be downloaded from
https://www.solarsystemscope.com/textures/

Only albedo textures are supported for now.
Each file has to be named <planetname>_albedo.jpg

This directory should contain:
* sun_albedo.jpg
* mercury_albedo.jpg
* venus_albedo.jpg
* earth_albedo.jpg
* jupiter_albedo.jpg
* saturn_albedo.jpg
* uranus_albedo.jpg
* neptune_albedo.jpg

* background.jpg
